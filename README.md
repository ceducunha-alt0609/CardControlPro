# Aurum Pro

Projeto pronto para publicar no GitHub Pages.

## Estrutura
- `index.html`
- `manifest.json`
- `sw.js`
- `icons/`
- `screenshots/`

## Publicação rápida no GitHub Pages
1. Crie um repositório novo no GitHub.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Em **Settings > Pages**, selecione a branch principal e a pasta `/root`.
4. Aguarde a publicação.

## Observações
- Os assets locais (`icons/` e `screenshots/`) foram incluídos como placeholders prontos para deploy.
- As bibliotecas externas continuam sendo carregadas por CDN, como no arquivo original.
- Se quiser, você pode substituir os placeholders por artes finais depois.
